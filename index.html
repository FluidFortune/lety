How This Works

Pisces Moon IDE is a live development environment for the Pisces Moon OS running on the LilyGO T-Deck Plus and (theoretically) the KodeDot.

Two Modes

  • Built-in App — A .cpp file that compiles into the OS. Wired into apps.h and launcher.cpp on export.
  • ELF Module — A standalone module loaded from the SD card at runtime. Compiled separately, drops into ./sd/apps/.

Live Preview

Your code runs in a JavaScript interpreter that emulates the Pisces Moon HAL exactly. The same source compiles for real hardware via PlatformIO.

Supported in preview:

  • Display: gfx->fillScreen, fillRect, drawRect, drawLine, drawCircle, fillCircle, setCursor, setTextColor, setTextSize, print, println, printf.
  • Touch: get_touch(&tx, &ty).
  • Trackball: update_trackball(), update_trackball_game().
  • Keyboard: get_keypress().
  • Timing: delay(ms), millis(), yield().
  • Theme: all C_* color constants.

Not in preview, but compiles for hardware:

  • WiFi, BLE, LoRa, GPS
  • SD card I/O
  • Audio
  • Custom libraries

Build & Flash to Hardware

The ⚡ Build & Flash button is the moonshot — it sends your source to the cloud build service, returns a compiled .bin, and flashes it directly to your T-Deck via Web Serial.

Requires: Chrome, Edge, or Opera on desktop. T-Deck connected via USB-C. No PlatformIO install needed on your machine.

Export

The Export button generates the final .cpp with proper AGPL header, plus a sidecar integration.txt with exact instructions for adding to apps.h and launcher.cpp.
